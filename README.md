# Brasileires

**Brasileires** é uma linguagem construída open-source baseada no português brasileiro, projetada como uma evolução sistemática, consistente e versionada da língua.

Este projeto não busca substituir o português, mas atuar como um **override linguístico**: o português é o estado padrão, enquanto o brasileires define regras que o modificam de forma previsível, lógica e intencional.

---

## 🧠 Filosofia

Brasileires não é apenas uma língua, é um **experimento de design linguístico**.

A proposta central é tratar a linguagem como um sistema que pode ser:

* analisado
* versionado
* debatido
* evoluído com governança clara

O foco não está na tradição, mas na **clareza, consistência e eficiência**.

---

## 📜 Axiomas

Os seguintes princípios guiam todas as decisões do projeto:

### 1. Consistência > Exceções

Regras devem ser previsíveis. Exceções são tratadas como falhas de design.

### 2. Fonêmica > Tradição

A escrita deve refletir a forma como as palavras são faladas, não necessariamente sua origem histórica.

### 3. Previsibilidade > Convenção

Mudanças devem seguir padrões claros e reproduzíveis.

### 4. Contexto resolve ambiguidade

Ambiguidades não devem ser eliminadas à força se o contexto já as resolve naturalmente.

### 5. Simplicidade estruturada

Reduzir complexidade sem perder capacidade expressiva.

### 6. Trade-offs são explícitos

Toda decisão tem custo. Nenhuma regra é adotada sem considerar seus impactos.

---

## 🔄 Modelo de Override

Brasileires não redefine tudo do zero.

Ele funciona como uma camada sobre o português:

```text
português (base)
   ↓
brasileires (override de regras)
```

Isso permite:

* evolução incremental
* comparação direta com o idioma original
* adoção gradual

---

## 🔤 Escopo do Projeto

O projeto é organizado em cinco áreas principais:

* **Ortografia** → como se escreve
* **Fonética** → como se fala
* **Gramática** → como se estrutura
* **Léxico** → quais palavras existem
* **Exemplos** → aplicação prática das regras

As regras completas estão na pasta [`docs/`](./docs).

---

## ⚖️ Trade-offs

Brasileires não busca perfeição absoluta.

Alguns compromissos inevitáveis incluem:

* perda de tradição histórica
* estranheza inicial na leitura
* necessidade de reaprendizado

Em troca, busca-se:

* maior regularidade
* aprendizado mais rápido
* menor carga cognitiva a longo prazo
