# Remoção do H

## Objetivo
Simplificar as palavras que possuam um "h" sem som desnecessário ou decorativo.

## Regra
Qualquer "h" sem som deve ser removido da palavra.

## Exemplos
* Homem -> Omem
* Hoje -> Oje
* Helicóptero -> Elicoptero
* Haver -> Aver

## Impaco
- Pode causar ambiguidades, mas é uma perda assegurada pelo princípio de homofonia e homografia;
- Simplifica a língua e remove letras decorativas.
